This document provides an explanation of tags used in faction_tags.csv and variant_tags.csv.
variant_tags.csv
note that variants not included in this file will not be modified by this mod

no - do not modify officer of ships with this variant
nr - do not remove ships with this variant from fleets
ca - officers of ships with this variant will be changed to cautious
ti - officers of ships with this variant will be changed to timid
st - officers of ships with this variant will be changed to steady
ag - officers of ships with this variant will be changed to aggressive
re - officers of ships with this variant will be changed to reckless
hs - officers of this ship variant will likely be given helmsmanship
ce - officers of this ship variant will likely be given combat endurance
im - officers of this ship variant will likely be given impact mitigation
dc - officers of this ship variant will likely be given damage control
fm - officers of this ship variant will likely be given field modulation
pd - officers of this ship variant will likely be given point defense
ta - officers of this ship variant will likely be given target analysis
bm - officers of this ship variant will likely be given ballistic mastery
se - officers of this ship variant will likely be given systems expertise
ms - officers of this ship variant will likely be given missile specialization
gi - officers of this ship variant will likely be given gunnery implants
ew - officers of this ship variant will likely be given energy weapon mastery
oe - officers of this ship variant will likely be given ordinance expert
pa - officers of this ship variant will likely be given polarized armour

faction_tags.csv
note that fleets from factions not in this file will not be modified by this mod (aside from receiving a tag)

no_autofit - with a few exceptions, fleets from factions with this tag will not be autofit
